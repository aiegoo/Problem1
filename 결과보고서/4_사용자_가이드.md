# 여행 사진 갤러리 사용자 가이드

## 🌟 시작하기

### 사이트 접속
1. 웹 브라우저에서 `http://localhost:3000` 방문
2. 또는 제공된 서버 URL 접속
3. 13개의 여행 사진 갤러리가 자동으로 로딩됩니다

### 지원 브라우저
- **Chrome** 88 이상 (권장)
- **Firefox** 78 이상
- **Safari** 14 이상  
- **Edge** 88 이상

## 🔍 주요 기능 사용법

### 1. 이미지 검색하기
![검색 기능](https://img.shields.io/badge/기능-검색-blue)

**사용 방법:**
1. 상단 검색창에 키워드 입력
2. "검색" 버튼 클릭 또는 Enter 키 입력
3. 일치하는 이미지만 화면에 표시됩니다

**검색 팁:**
- **단일 검색**: `"image"` → 모든 이미지 표시
- **다중 검색**: `"image 3"` → "Image 3" 이미지만 표시
- **대소문자 무관**: `"IMAGE"` 또는 `"image"` 동일 결과
- **빈 검색**: 검색창 비우고 검색 → 모든 이미지 복원

**예시:**
```
검색어: "image 5"     → "Image 5" 결과 1개
검색어: "description" → 설명에 "description" 포함된 모든 결과
검색어: ""           → 전체 13개 이미지 표시
```

---

### 2. 이미지 제목 복사하기
![복사 기능](https://img.shields.io/badge/기능-복사-green)

**사용 방법:**
1. 복사하고 싶은 이미지를 클릭
2. 자동으로 이미지 제목이 클립보드에 복사됩니다
3. 성공 메시지가 팝업으로 표시됩니다

**성공 메시지 예시:**
```
'Image 1'가 클립보드에 복사되었습니다.
'Image 10'가 클립보드에 복사되었습니다.
```

**활용 방법:**
- 복사된 제목을 다른 앱에 붙여넣기 (Ctrl+V)
- 메모장, 이메일, 채팅 등에 활용

---

### 3. 댓글 작성하기
![댓글 기능](https://img.shields.io/badge/기능-댓글-orange)

**사용 방법:**
1. 원하는 이미지 카드 하단의 댓글 입력창 클릭
2. 댓글 내용 입력
3. "Submit" 버튼 클릭
4. 댓글이 즉시 표시됩니다

**댓글 규칙:**
- **최대 5개**: 이미지당 5개 댓글만 작성 가능
- **빈 댓글 불가**: 내용 없이 제출 시 알림 표시
- **시간 표시**: 작성 시간이 자동으로 추가됩니다

**댓글 형식:**
```
안녕하세요! (1월 30일 14:38)
멋진 사진이네요! (1월 30일 14:40)
```

**제한 메시지:**
- 빈 댓글 제출 시: `내용을 입력해주세요.`
- 5개 초과 시: `하나의 이미지에는 5개의 댓글만 추가할 수 있습니다.`

---

### 4. 좋아요 누르기
![좋아요 기능](https://img.shields.io/badge/기능-좋아요-red)

**사용 방법:**
1. 이미지 카드의 "Likes" 버튼 클릭
2. 숫자가 1씩 증가합니다
3. 최대 5번까지 클릭 가능

**좋아요 표시:**
```
Likes: 0  → 클릭 → Likes: 1
Likes: 1  → 클릭 → Likes: 2
...
Likes: 5  → 클릭 → 제한 알림 표시
```

**제한 메시지:**
```
하나의 이미지에는 5번의 좋아요만 누를 수 있습니다.
```

---

### 5. 이미지 레이지 로딩
![성능 기능](https://img.shields.io/badge/기능-성능최적화-purple)

**자동 작동:**
- 페이지 로드 시 플레이스홀더 이미지 표시
- 스크롤하여 이미지가 화면에 나타날 때 실제 이미지 로딩
- 네트워크 트래픽 절약 및 빠른 초기 로딩

**사용자 경험:**
- 부드러운 이미지 로딩
- 빠른 페이지 응답
- 데이터 절약 효과

## 💡 사용 팁 및 요령

### 효율적인 검색 방법
1. **구체적인 키워드**: "3", "image 3" 등 명확한 검색어 사용
2. **공백 활용**: 여러 단어로 정확한 결과 찾기
3. **빈 검색으로 초기화**: 모든 이미지 다시 보기

### 댓글 작성 팁
1. **의미 있는 댓글**: 구체적이고 유용한 피드백 작성
2. **5개 제한 고려**: 중요한 댓글부터 우선 작성
3. **시간 활용**: 타임스탬프로 댓글 순서 확인

### 좋아요 전략
1. **신중한 선택**: 5번 제한을 고려한 선별적 좋아요
2. **다양한 이미지**: 여러 이미지에 골고루 좋아요
3. **즉시 반영**: 클릭 후 바로 확인되는 피드백

## 🚨 문제해결 가이드

### 자주 묻는 질문

**Q: 이미지가 모두 똑같아 보여요**
A: 이는 정상적인 현상입니다. 제공된 이미지 파일들이 유사한 내용을 담고 있어서 시각적으로 비슷해 보이지만, 실제로는 다른 파일들이 로딩되고 있습니다. 이미지 제목("Image 1", "Image 2" 등)으로 구분할 수 있습니다.

**Q: 검색이 잘 안 돼요**
A: 다음을 확인해주세요:
- 정확한 키워드 입력 (예: "image 3")
- 대소문자는 상관없음
- 공백 구분으로 여러 단어 검색
- 빈 검색으로 전체 결과 복원

**Q: 클립보드 복사가 안 돼요**
A: 브라우저 설정을 확인해주세요:
- HTTPS 또는 localhost 환경 필요
- 브라우저 권한 허용
- 팝업 차단 해제

**Q: 댓글/좋아요가 저장되지 않아요**
A: 현재 버전은 세션 기반 저장입니다:
- 페이지 새로고침 시 초기화됨
- 브라우저 종료 시 데이터 삭제
- 향후 서버 연동 시 영구 저장 예정

### 브라우저별 문제해결

**Chrome/Edge:**
- 일반적으로 모든 기능 완벽 지원
- 개발자 도구(F12)로 에러 확인 가능

**Firefox:**
- 클립보드 권한 요청 시 허용 선택
- about:config에서 dom.events.asyncClipboard.enabled 확인

**Safari:**
- 보안 설정에서 JavaScript 활성화 확인
- 개인정보 보호 설정 확인
