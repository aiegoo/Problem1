# 여행 사진 갤러리 사이트 구축하기

각 항목의 요구사항에 맞게 기능을 구현해 주세요.

###

## ✅ 1. 이미지 카드 나타내기

> 이미지 카드를 화면에 나타나도록 구현하는 문제입니다.

![demo1.png](https://grepp-programmers.s3.ap-northeast-2.amazonaws.com/files/production/7025108b-7949-416a-bba8-fed940cd7afe/demo1.png)

- 이미지 카드는 아래 7가지 요소로 구성되어 있습니다.

  - 이미지

  - 이미지 제목

  - 이미지 설명

  - `Likes` 버튼

  - 댓글

  - 댓글 입력창과 `Submit` 버튼

- 아래의 마크업 정보를 활용하여 화면에 13개의 이미지 카드를 나타냅니다.

  ```html
  <div class="image-card" data-id="1">
    <img
      data-src="./data/images/image1.jpg"
      src="./data/images/image1.jpg"
      alt="Image 1"
      class=""
    />
    <h3>Image 1</h3>
    <p>Description of image 1</p>
    <button class="like-btn">Likes: 3</button>
    <div class="comments">
      <li>amazing! <span class="comment-timestamp">(2월 12일 10:01)</span></li>
    </div>
    <form class="comment-form">
      <input type="text" placeholder="Add a comment" />
      <button type="submit">Submit</button>
    </form>
  </div>
  ```

###

## ✅ 2. 이미지 검색하기

> 이미지를 검색하는 기능을 구현하는 문제입니다.

![demo2.gif](https://grepp-programmers.s3.ap-northeast-2.amazonaws.com/files/production/df0cb84b-2651-4557-bdc9-66103d43018b/demo2.gif)

###

- 이미지 검색창에 임의의 문자를 입력 후 검색 버튼을 클릭하면 이미지 제목 혹은 설명에 검색한 문자가 포함된 이미지가 나타납니다.

- 이미지 검색 시 띄어쓰기 구분하여 검색합니다.

- 일치하는 결과가 없을 시 빈 화면이 나타납니다.

  - 예를 들어, `image 3`을 입력하여 검색하면 `Image 3`이미지가 나타납니다.

###

## ✅ 3. 이미지 복사하기

> 이미지를 복사하는 기능을 구현하는 문제입니다.

![demo3.gif](https://grepp-programmers.s3.ap-northeast-2.amazonaws.com/files/production/bbbc773f-4d2e-4db7-aa9f-9533f1475191/demo3.gif)

###

- 임의의 이미지를 클릭하면 아래와 같이 동작합니다.

  - 이미지의 제목이 클립보드에 복사됩니다.

  - `'(이미지 제목)'가 클립보드에 복사되었습니다.` 문구의 다이얼로그 창(`window.alert`)이 나타납니다.

- 예를 들어, `Image 1` 제목의 이미지를 클릭하면 Image 1이 클립보드에 복사되며, `'Image 1'가 클립보드에 복사되었습니다.` 문구의 다이얼로그 창이 나타납니다.

###

## ✅ 4. 댓글 작성하기

> 이미지에 댓글을 남기는 기능을 구현하는 문제입니다.

![demo4.gif](https://grepp-programmers.s3.ap-northeast-2.amazonaws.com/files/production/ffbf3eef-1d05-4cee-a7d3-fe1b16fd85ca/demo4.gif)

###

- 이미지 카드의 `Submit` 버튼을 댓글 입력 없이 클릭하면 `내용을 입력해주세요.` 문구의 다이얼로그 창(`window.alert`)이 나타납니다.

- 임의의 이미지 카드에서 댓글을 입력 후 `Submit` 버튼을 클릭하면 아래와 같이 동작합니다.

  - 댓글 입력창은 초기화되고, 작성한 댓글이 댓글 영역에 나타납니다.

  - 작성된 댓글에는 `(O월 O일 00:00)` 형태의 시간 정보가 포함되어야 합니다.

    - 예를 들어, 4월 13일 오후 2시 38분에 댓글을 작성하였다면 `댓글 (4월 13일 14:38)`와 같이 댓글 정보가 포함되어야 합니다.

- 하나의 이미지 카드에 5개의 댓글만 작성이 가능하며, 댓글은 예시 GIF처럼 등록순으로 나열됩니다.

  - 이미지 카드에 5개의 댓글이 모두 작성되어 있을 때 추가로 댓글을 작성하는 경우 `하나의 이미지에는 5개의 댓글만 추가할 수 있습니다.` 문구의 다이얼로그 창이 나타납니다.

## ✅ 5. 좋아요 누르기

> 이미지에 좋아요를 누르는 기능을 구현하는 문제입니다.

![demo5.gif](https://grepp-programmers.s3.ap-northeast-2.amazonaws.com/files/production/06bb0e51-a5cd-41ea-bf24-f007164192be/demo5.gif)

###

- 이미지 카드의 `Likes` 버튼을 클릭하면 `Likes` 버튼의 숫자가 1씩 증가한다.

- `6`번 이상 `Likes` 버튼을 클릭하면 `하나의 이미지에는 5번의 좋아요만 누를 수 있습니다.` 문구의 다이얼로그 창(`window.alert`)이 나타난다.
