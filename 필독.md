## 📌 응시 환경 안내

> 더 자세한 내용은 상단 네비게이션 바의 도움말 버튼 클릭 시 나타나는 문서를 참고해 주세요.

### 1. 기본 준비

- VS Code 화면 상단에서 `터미널` -> `새 터미널`을 클릭하여 터미널을 열어주세요.

- 키보드 단축키로도 손쉽게 열고 닫을 수 있습니다.
  - control + `(백틱)

### 2. 웹 서버 실행하기

#### 웹 서버 실행

웹 서버 프로세스를 실행하기 위해 터미널에 아래와 같은 명령어를 순차적으로 입력합니다.

```bash
$ npm install
$ npm start
```

#### 실행 확인

아래의 두 가지 방법으로 실행을 확인할 수 있습니다.

1. 자동으로 나타나는 IDE 내의 `Simple Browser`에서 실행 결과 화면을 확인할 수 있습니다.

2. 터미널에 출력되는 `http://localhost:포트번호` 형식의 주소를 클릭했을 때 열리는 브라우저 창에서 실행 결과 화면을 확인할 수 있습니다.

   - 서버가 정상적으로 실행되면 터미널에 `http://localhost:포트번호` 형식의 주소가 출력됩니다.

   - 이 주소는 자동으로 링크로 인식되며, macOS에서는 `Command (⌘)` 키, Windows/Linux에서는 `Ctrl` 키를 누른 상태에서 주소를 클릭하면 브라우저로 바로 접속하여 웹 서버 실행을 확인할 수 있습니다.

   - 예:

     - macOS: `⌘` + 클릭

     - Windows: `Ctrl` + 클릭

### 3. 코드 채점하기

우측 상단의 `코드 채점` 버튼을 클릭하여 채점을 진행하고, `채점 로그`에서 채점 결과를 확인합니다.

`코드 채점`은 아래와 같이 진행됩니다.

- 코드 채점은 **최고점이 아닌 마지막 코드 채점 결과를 기준으로 점수가 반영**됩니다.

- 각 문제에 대해서 **응시 종료 전까지 최소 1번은 `코드 채점`을 진행**해야 합니다.

- 만약 응시 시간 내에 `코드 채점`을 진행하지 않은 경우 종료 시 강제로 `코드 채점`이 진행됩니다.

  - 이때 컴파일 에러 등이 발생한 경우 0점 처리가 될 수 있으니 유의바랍니다.

### 4. 테스트 종료하기

`테스트 종료` 버튼을 누르면 모든 시험이 종료됩니다.

- `코드 채점`을 통해 얻은 마지막 결과를 기준으로 테스트가 종료됩니다.

- `테스트 종료` 이후에는 시험에 다시 응시할 수 없습니다.

## 📌 코드 작성 안내

### ✅ 유의 사항

- `JavaScript`로 과제를 수행해야 합니다.

- 문제 풀이 과정에서 검색이 가능하지만 시험 상황에 따라 규정이 변경될 수 있습니다.

- 문제 풀이 환경 내에 제공된 파일들의 이름과 구조를 임의로 수정하거나 삭제하지 않습니다.

  - 채점 가능

    - `script.js` 파일에 모든 솔루션 코드를 작성하는 경우

      ```javascript
      // script.js

      console.log("Hello World");
      ```

    - 추가적인 파일을 생성하여 해당 코드를 `script.js`에 연결하는 경우

      ```javascript
      // script.js

      import App from "./src/App.js";

      new App(document.querySelector("body"));
      ```

      ```javascript
      // src/App.js

      console.log("Hello World");
      ```

  - 채점 불가능

    - HTML 문서 내에 미리 작성한 `<script>` 안에 정답 코드를 작성하는 경우
      ```html
      <body>
        ...
        <script>
          console.log("Hello World");
        </script>
      </body>
      ```
    - `script.js`가 아닌 다른 이름의 `.js` 파일을 만들어 코드를 작성하는 경우

      ```javascript
      // app.js
      console.log("Hello World");
      ```

- 문제 풀이 시 반드시 제공된 레이아웃 구성 및 선택자를 그대로 활용해 주세요.

  - 주어진 레이아웃 구성 및 선택자를 **응시자가 임의로 변경할 경우** 정상적으로 채점이 되지 않을 수 있습니다.

  - 테스트 형식에 벗어나는 형태는 오답 처리되니 지문을 읽고 정상적으로 채점이 될 수 있도록 코드를 작성해 주세요.

  - 단, 해당 문항의 경우 `index.html`에 제공된 기본 코드를 수정하는 것은 가능합니다.

**이후 `문제 지문 탭`의 문제 지문을 잘 읽고 요구사항을 해결합니다.**
